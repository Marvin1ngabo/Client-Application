# School Management System - Client Frontend

React frontend for parents and students.

## Tech Stack

- React 18 + Vite
- React Router v6
- TailwindCSS
- Axios
- Zustand (state management)
- React Query
- React Hot Toast

## Installation

```bash
npm install
```

## Environment Setup

```bash
cp .env.example .env
```

## Development

```bash
npm run dev
```

Runs on http://localhost:3000

## Build

```bash
npm run build
```

## Features

- Login/Register with device verification
- Dashboard with fee balance, grades, attendance
- Responsive design (mobile + desktop)
- Toast notifications
- Protected routes
